# F12.1 - Organization Management

## Overview

Organization, team, member, and invitation management with role-based access control and plan-based limits.

## Architecture

```
Client/lib/organization/
  org-management.ts        # Orgs, teams, members, invitations, RBAC
  org-management.test.ts   # 82 tests
```

## Features

| Function | Purpose |
|----------|---------|
| createOrganization | Create org with auto-slug, plan-based settings |
| generateSlug | URL-safe slug from name |
| canPerformAction | Role hierarchy authorization check |
| createInvitation | 7-day expiry invitation with pending status |
| isInvitationValid | Expiry and status validation |
| addMemberToOrg | Add member with plan limit enforcement |
| removeMember | Remove member (owner protected) |
| createTeam | Team creation with optional members |
| getTeamMembers | Filter members by team |
| getMemberRole | Lookup user role in org |
| updateMemberRole | Role update with privilege checks |
| getOrgStats | Member/team counts and utilization |

## Plan Limits

| Plan | Members | Teams | Specs |
