# F11.4 - AI Review Assistant

## Overview

Automated review system for specifications and code, checking completeness, security, performance, naming, and accessibility with severity-weighted scoring.

## Architecture

```
Client/lib/ai/
  ai-review.ts        # Review rules, scoring, formatting
  ai-review.test.ts   # 88 tests
```

## Review Categories

| Category | Checks |
|----------|--------|
| Completeness | Missing title, no acceptance criteria, short content, no user story, missing sub-sections |
| Security | Unencrypted passwords, SQL injection, hardcoded credentials, API key exposure, missing auth |
| Performance | Unbounded queries, missing pagination, no caching, synchronous large-data ops, no rate limiting |
| Naming | Single-character variables, unclear abbreviations, inconsistent casing, generic names |
| Accessibility | Missing ARIA, no keyboard navigation, missing contrast, no screen reader support |

## Severity Weights

| Severity | Weight |
|----------|--------|
| critical | 10 |
| warning | 5 |
| info | 2 |
| suggestion | 1 |

## Functions

conductReview, formatFinding, generateReviewSummary, filterBySeverity, sortFindings

## Score Calculation

Score = 100 minus sum of (finding weight per severity), clamped to 0 minimum.
