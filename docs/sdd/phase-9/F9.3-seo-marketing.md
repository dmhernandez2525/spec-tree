# F9.3 - SEO & Marketing Site

## Overview

SEO metadata management, Open Graph tag generation, JSON-LD structured data, sitemap/robots.txt generation, and privacy-respecting analytics tracking.

## Architecture

```
Client/lib/seo/
  metadata.ts             # OG tags, JSON-LD, sitemap, robots.txt
  analytics-tracking.ts   # Event tracking with DNT respect
  seo.test.ts             # 44 tests
```

## Features

- **Page Metadata**: Default SEO config, page title builder, Open Graph tag generation
- **Structured Data**: JSON-LD for Organization, WebApplication, and FAQ schemas
- **Sitemap**: XML sitemap generation with priority, changefreq, lastmod
- **Robots.txt**: Configurable allow/disallow rules with sitemap directive
- **Analytics**: Event queuing with category tracking, flush to API, DNT respect, anonymous user IDs
